# Синхронизация товаров заказа и сделки

## Синхронизация Битрикс -> амо
Для того, чтобы передавать товароы из заказа в сделку amoCRM, необходимо на вкладке «Сделка» правила интеграции отметить галочку «Передавать товары». Галочка активна только для правил интеграции событий, связанных с заказом и при выключенном «неразобранном».

## Синхронизация amo CRM -> Bitrix
Для корректной синхронизации количества товаров необходимо:
1. Выполнить пункты 1 и 2 [общих настроек](./status-sync.md#общие-настройки) синхронизации статусов
2. Там же отметить «amoCRM > интернет-магазин»
3. Добавить (если еще не добавлен) со стороны amoCRM вебхук на адрес, указанный в пункте «URL для вебхука» и настроить его срабатывание на событие «Сделка изменена» для сделки.
