# beforeMakeCatalogElementByBasketItem (с версии 2.6.9)
Вызывается перед созданием товара для амо на основе товара из корзины. Есть возможность создать свой товар, вернув его вторым параметром, в этом случае штатное создание запущено не будет. Либо вернуть `false`, в этом случае штатное создание тоже запущено не будет.

## Параметры
0. `Bitrix\Sale\BasketItem` — товар в корзине;

## Обработка события
Аналогично обработке [onBeforeAmoPush](./onbeforeamopush.md).
