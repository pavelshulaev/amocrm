# beforeSearchOneByBasketItem (с версии 2.6.9)
Вызывается перед поиском товара в amoCRM на основе товара из корзины. Есть возможность реализовать с его помощью свой поиск, вернув товар из Амо третьим параметром, или отменить штатный, вернув `false`.

## Параметры
0. `Bitrix\Sale\BasketItem` — товар в корзине;
1. `integer` — id каталога в амо срм.

## Обработка события
Аналогично обработке [onBeforeAmoPush](./onbeforeamopush.md).
